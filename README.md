# SICA-1a-Unity-project
The project is based on Unity 2022.3.18f1, this game will be a First Person Shooter multiplayer.

Objectif of the project:

- PlayerMovement
- ShootingSystem
- HitSystem
- RespawnSystem
- HUD
- WeaponSystem
- MultiplayerSystem
- PauseMenu
- ScoreSystem
- ReloadSystem
- HealSystem
- AmmoSystem
- UserInformationSystem
- Drop/PickupSystem
- MapSystem
- SoundDesign
- DedicatedServer
